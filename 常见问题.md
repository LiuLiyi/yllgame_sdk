## YGSDK 对接常见问题

### 1.登陆界面游戏切换到后台，再切回前台，游戏背景黑屏
**原因**：因为登陆界面会暂停游戏的UI绘制  
**解决方案**：在游戏的Activity的Onresume方法中，重新激活游戏的焦点
```java
    @Override
    protected void onRestart() {
        super.onRestart();
        onWindowFocusChanged(true);
    }
```
